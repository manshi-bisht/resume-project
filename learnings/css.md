# CSS Learnings

## What is CSS?
 CSS (Cascading Style Sheets) is used to style HTML webpages and improve their appearance.

 CSS is used to change the look and feel of a webpage.
 CSS makes the website more presentable and attractive.

## Types of CSS
  - Inline CSS
  - Internal CSS
  - External CSS

```
- External CSS is the best method because it keeps HTML and styling separate.
- Inline CSS is less popular because they make websites incredibly difficult
 to maintain and break the concept of code reusability.
```
## CSS Selectors
    -Universal Selector (*) 
    -Class Selector(.classname) 
    -id  Selector(#idname)
    -Element Selector(p,h1,etc)

## CSS Properties
 We can style text using properties like:
  - color
  - font-size
  - font-family
  - font-weight
  - background-color.
  - Margin and padding are used to create spacing.
  - Border and border-radius help create attractive boxes and rounded corners.
 - CSS can align text and images.
 - The :hover pseudo-class changes an element's appearance when the mouse pointer is over it.
 - Navigation bars and buttons become more interactive using hover effects.

## Conclusion

CSS makes webpages attractive, readable, and user-friendly. It separates design from HTML structure, making websites easier to maintain and update.
